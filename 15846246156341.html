两数之和问题

2020/03/19 posted in  算法&数据结构

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例:
给定 nums = [2, 7, 11, 15], target = 9
因为 nums[0] + nums[1] = 2 + 7 = 9
所以返回 [0, 1]

这个题目的在 OC 里的解法就是用字典,
首先创建一个字典,value 为数组的 index,key 为数组元素,
例如:数组[2,7,11,15];
那么这个字典就是
@{@(0):@"2",
@(1):@"7",
@(2):@"11",
@(3):@"15",
}
比如目标和为 9, 那么组合的话,第一个为 2,那么 "9-2" 作为 key,去字典里查找,看能否找到目标值,如果这个字典里有符合的 肯定能找到 value,反之就是没有

代码如下:


- (NSArray *)getTwoNumSum:(int)targetSum array:(NSArray *)arr {
    NSMutableDictionary *tmpDic = [NSMutableDictionary dictionary];
    NSMutableArray *tmpA = [NSMutableArray array];
    for (int i = 0; i < arr.count; i++) {
        [tmpDic setValue:@(i) forKey:arr[i]];
    }
    
    [arr enumerateObjectsUsingBlock:^(id  _Nonnull obj, NSUInteger idx, BOOL * _Nonnull stop) {
        NSString *keyString = [NSString stringWithFormat:@"%d",targetSum - ((NSString *)obj).intValue];
        NSNumber *lastIndex = tmpDic[keyString];
        if (lastIndex && idx < lastIndex.integerValue) {
            [tmpA addObject:@([lastIndex integerValue])];
            [tmpA addObject:@(idx)];
        }
    }];
    
    return tmpA;
    
}
